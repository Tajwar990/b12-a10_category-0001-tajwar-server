This is the **backend API** for the Book Haven web application. It provides CRUD operations for managing books, fetching latest books, and retrieving user-specific books. The backend is built with **Node.js**, **Express**, and **MongoDB Atlas**.

## ðŸ›  Features

- **CRUD Operations**:
  - **Create**: Add a new book.
  - **Read**: Fetch all books or a specific book by ID.
  - **Update**: Update a book by ID.
  - **Delete**: Remove a book by ID.
- **User-Specific Books**: Fetch books added by a specific user using their email.
- **Latest Books**: Fetch the latest 6 books for homepage display.
- **CORS Enabled**: Allows frontend apps to access the API securely.
